# MzzUtils
*首次提交 里面包含AES加密解密、Activity管理栈、文件工具类以及各种获取intent意图*
### Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
```
### Add the dependency
```
dependencies {
	        implementation 'com.github.Mazhanzhu:MzzUtils:Tag'//最新 1.0.0
	}
  ```
